# 🌱 손목 정원 (Wrist Garden)

웹캠으로 손동작을 인식해 즐기는 브라우저 기반 손목 운동 게임입니다.
카메라 앞에서 손을 움직이고 손가락을 오므리는(핀치) 동작으로 정원을 가꿉니다.

## ✨ 특징

- **웹캠 손 인식** — 별도 설치 없이 브라우저에서 카메라로 손동작을 추적
- **핀치 제스처** — 엄지·검지를 오므려 상호작용
- **콤보 / 점수 / 미터** 시스템으로 재미있는 피드백
- **전체화면 지원** 및 반응형(16:9) 레이아웃
- **한국어 UI**, 아기자기한 정원 테마 디자인

## 🚀 실행 방법

1. 저장소를 클론하거나 `wrist-garden_44.html` 파일을 내려받습니다.
2. HTML 파일을 최신 브라우저(Chrome 권장)에서 엽니다.
3. 카메라 사용 권한을 허용합니다.
4. 카메라 앞에서 손을 움직여 플레이합니다.

> 카메라 접근은 보안상 `https` 또는 `localhost` 환경에서 동작합니다.
> 로컬에서 실행 시 간단한 로컬 서버를 사용하는 것을 권장합니다.
>
> ```bash
> # 예: Python 내장 서버
> python -m http.server 8000
> # 브라우저에서 http://localhost:8000/wrist-garden_44.html 접속
> ```

## 🛠 기술

- 순수 HTML / CSS / JavaScript (단일 파일)
- 웹캠 및 손 추적 기반 인터랙션
