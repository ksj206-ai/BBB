# 🌱 손목 정원 (Wrist Garden)

웹캠으로 손과 팔을 인식해, **손목 움직임으로 정원을 가꾸는** 브라우저 기반 재활·운동 게임입니다.
MediaPipe로 손·팔 관절을 추적하고, 손목의 굽힘/폄·좌우 편위·핀치·악력 같은 동작을
각각 다른 미니게임 장르로 만들어 **"지루한 재활을 게임처럼 꾸준히"** 하도록 유도합니다.

> **웰니스 포지셔닝** — 진단/치료 목적이 아니라 **예방·가동성(ROM) 관리·습관 형성**을 돕는 도구입니다.

---

## ✨ 주요 특징

- 🎥 **웹캠 손·팔 인식** — 별도 설치 없이 브라우저에서 MediaPipe로 손동작 추적
- 🎮 **운동별 미니게임** — 운동마다 다른 게임 장르로 재미있게
- 📐 **ROM 측정 시스템** — MediaPipe로 손목 가동범위(편위·굴곡/신전) 측정
- 🤸 **보상동작(Compensation) 감지** — 팔꿈치·어깨가 움직이면 경고 (`COMP_TH=0.48`)
- 🌈 **운동별 전용 배경** — 코드로 직접 그린 아기자기한 정원 씬 (이미지 파일 0개)
- 🏆 **콤보 / 점수 / 진행 게이지 / 파티클** 피드백
- 💾 **localStorage 저장** — 진행 상황·식물 성장 기록
- 🖥️ **전체화면 지원**, 반응형(16:9) 레이아웃, 한국어 UI

## 🕹 운동 목록 (7종 중 6종 완성)

| 상태 | 운동 | mode | 게임 장르 | 인식 동작 |
|:---:|------|------|-----------|-----------|
| ✅ | ✋ 손 풀기 | `glide` | 3레인 리듬게임 | 손가락 쫙 / 갈고리 / 주먹 |
| ✅ | 🔨 밭 망치는 두더지 잡기 | `dig` | 2줄 두더지 잡기 | 손목 굽힘 / 폄 |
| ✅ | 💧 촉촉하게 물 주기 | `rain` | 좌우 러너 | 손목 좌우 편위 |
| ✅ | 🍓 잘 익은 열매 따기 | `pinch` | 열매 수확 (집고 1.5초 유지) | 엄지-검지 핀치 |
| ⏳ | ☀️ 햇빛 모으기 | `stretch` | (게임화 다듬는 중) | 손가락/손목 펴기 |
| ✅ | 🦋 몰려드는 벌레 쫓기 | `grip` | 반응 디펜스 | 주먹(딱정벌레) / 손바닥(잎벌레) |
| ✅ | 💉 영양제 주기 | `isopump` | 등척성 파워존 유지 | 주먹 쥐고 힘 조절 유지 |

### 완성된 게임 상세

- **✋ 손 풀기 (`glide`)** — 3개 레인으로 내려오는 노트를 손가락 모양(쫙/갈고리/주먹)으로 맞추는 리듬게임.
- **🔨 두더지 잡기 (`dig`)** — 2줄×3구멍 두더지밭. 윗줄=점프 두더지(손목 **폄**), 아랫줄=빼꼼 두더지(손목 **굽힘**). 망치가 자동으로 이동하고, 잡으면 별이 뱅글 도는 연출. **전용 흙밭 배경**(밭고랑·나무울타리·허수아비·작물·고추잠자리 2마리).
- **💧 물 주기 (`rain`)** — 손목을 좌우로 기울여 물뿌리개를 움직이며 💧는 받고 🍂마른 잎은 피하는 러너. **전용 "비 갠 정원" 배경**(무지개·물웅덩이·수국·물안개).
- **🍓 열매 따기 (`pinch`)** — 반짝이는 잘 익은 열매를 엄지·검지로 집어 **1.5초 유지**하면 수확. 바구니에 열매가 쌓임.
- **🦋 벌레 쫓기 (`grip`)** — 화면 위에서 **벌레가 새싹을 향해 날아옴**. 🪲딱정벌레(단단)는 ✊주먹으로, 🐛잎벌레(무름)는 🖐️손바닥으로 쳐냄. 도달 게이지가 차기 전에 올바른 손 모양으로 반응. 놓치면 새싹이 흔들리고 콤보 초기화. 시도가 진행될수록 반응 시간이 짧아지는 난이도 상승.
- **💉 영양제 주기 (`isopump`)** — **주먹을 적당히 꽉** 쥐어 **초록 파워존**을 유지하면 영양액이 **관을 타고 뿌리로 스며들고**, 3초 유지마다 새싹이 **쑥 자람**. 너무 세게 쥐면(과부하) 물주머니가 떨리며 흐름이 멎음. 손목 중립은 강제하지 않고 부드러운 힌트만 제공(주먹만으로 인식). 손목 주변 **등척성 근력**을 조절된 강도로 케어.

## 🎨 코드 구조

바닐라 JS + Canvas 2D. 외부 이미지 없이 **모든 그래픽을 코드로 그림**. 3개 파일로 구성:

```
index.html   화면 구조 — <link href="style.css"> + <script type="module" src="game.js">
style.css    모든 스타일 (디자인·HUD·버튼, 폰트 @import)
game.js      게임 로직 전부 (약 1,160줄)
├── 헬퍼        addPart / burst / popText 등 (파티클·텍스트 연출)
├── 배경        drawScene(기본 정원) · drawMoleField(흙파기) · drawRainField(물주기)
├── 요소        pestBug(벌레) 등 캔버스 드로잉
├── 측정 시스템  MediaPipe 기반 ROM(가동범위) 측정
├── 운동 게임    s.mode==="glide" / "dig" / "rain" / "pinch" / "grip" / "isopump" … 로 분기
└── 게임 루프    requestAnimationFrame
```

**주요 상수** (`W=960, H=540, groundY=348`), 보상동작 임계값 `COMP_TH=0.48`.
MediaPipe(`@mediapipe/tasks-vision`)는 game.js에서 CDN ES 모듈로 import.

## 🚀 실행 방법

카메라 접근은 보안상 **`https` 또는 `localhost`** 환경에서만 동작합니다. 로컬에서는 간단한 서버를 띄워 실행하세요.

```bash
# 저장소 클론
git clone https://github.com/ksj206-ai/BBB.git
cd BBB

# 로컬 서버 실행 (Python 내장 서버 예시)
python -m http.server 8000

# 브라우저에서 접속 (Chrome 권장)
#   http://localhost:8000/index.html   (또는 http://localhost:8000/)
```

접속 후 **카메라 권한을 허용**하고, 카메라 앞에서 손을 움직여 플레이합니다.

## ⚠️ 개발 메모 / 주의사항

- **폄(신전) 인식은 2D 웹캠 한계로 약함** → 관련 게임의 목표 각도를 낮게 설정.
- **보상동작 감지** `COMP_TH=0.48` — 손목만 움직이도록 팔꿈치·어깨 이동을 감지해 경고.
- 각 게임 로직은 `s.mode==="dig"/"pinch"/"rain"` 등으로 찾을 수 있음.
- 배경 함수: `drawScene`(기본) · `drawMoleField`(흙파기 전용) · `drawRainField`(물주기 전용).
- **편집 후 문법 검사 권장** (`<script>` 추출 후 `node --check`).
